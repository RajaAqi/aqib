
- Click **"Select"** button on any template

1. Click **"Back to Editor"**
2. Fill in your personal information
3. Add experiences, education, skills
4. Changes are auto-saved

### Step 6: Preview & Export
1. Click **"Preview"** from editor
2. See your resume in selected template
3. Click **"Export as PDF"** to download

## üìù Data Entry Tips

### Personal Information
```
Full Name: Your complete name
Job Title: Current role or target role
Phone: Include country code if international
Email: Professional email address
Address: City, State (or Country)
Summary: 2-3 sentences about your background
```

### Experience
```
Title: Your job title
Company: Company name
Start: Month Year (e.g., "Jan 2022")
End: Month Year or "Present"
Bullets: Key achievements (3-5 per role)
```

### Education
```
Degree: B.Tech, MBA, etc.
Institute: University/School name
Year: Graduation year
```

### Skills
```
- List as individual tags
- Press Enter after each
- 6-10 skills recommended
- Tech-focused or general
```

### Languages
```
- List as individual items
- Include proficiency if needed
- Press Enter after each
```

## üñ®Ô∏è Export Tips

### For Online Applications
1. Export as PDF (recommended format)
2. Use modern browser (Chrome/Edge)
3. Check file size (ideally under 3MB)
4. Verify colors display correctly

### For Email
1. PDF format works best
2. Classic/Minimalist: Standard PDF
3. Modern/Dark: Color PDF preferred
4. Include as attachment

### For Printing
1. **Color Printer Available?**
   - YES: Use Dark Modern or Modern Sidebar
   - NO: Use Classic, Minimalist, or Two-Column

2. **Standard 8.5" x 11" Paper**
   - All templates fit perfectly
   - Check print preview first

3. **Before Printing**
   - Preview in browser (Ctrl+P or Cmd+P)
   - Check formatting
   - Verify all text appears

## üêõ Troubleshooting

### PDF Export Not Working?
- Check browser console for errors
- Try a different browser
- Ensure all fields are filled
- Wait for preview to render

### Template Looks Different When Exported?
- Preview in print mode first
- Check browser print settings
- Ensure margins are correct
- Use PDF instead of image

### Content Missing in Preview?
- Save resume first (auto-save works)
- Refresh page
- Check that data was entered correctly
- Verify no empty required fields

### Template Not Saving?
- Check browser storage/cookies enabled
- Clear browser cache
- Try private/incognito mode
- Check disk space
